Service prices

- Раздел содержит информацию о ценах на услуги.

- Поиск осуществляется по модели телефона.

Telephones

- Основная информация хранится в базе данных раздела.

- Регистр - номер регистрации в базе данных. Он состоит из букв и цифровой части.

  В разделе Settings обе части необходимо указать перед самым первым началом

  пользования программой. Цифровая часть регистрации увеличивается автоматически

  на 1 (от максимального значения в памяти) при каждой новой записи.

  По превышении 99999 - обнуляется.

- Стикер - номер регистрации сервиса.

  Указывается при добавлении/редактировании записи в произвольном порядке.

- Аналогично вводится остальная информация.

- Заполнение всех окон формы является обязательным.

- При наличии информации о номере телефона или адресе электронной почты

  клиента (а также в Settings "Разрешить отправку сообщений" - в положении "Yes")

  в режиме edit активируется ссылка Send message.

  Дальнейшие действия по этой ссылке описаны в разделе messages.

- Поиск осуществляется по модели телефона, номеру регистрации, номеру стикера,

  неисправности, статусу, комментариям,IMEI, номеру телефона клиента.

Details

- Содержится информация о деталях, имеющихся в наличии в данном сервисе.

- При вводе/редактировании запись содержит позицию "Показать всем" - показать всем

  пользователям сайта ( в таблице отображается в столбце "Вид").

- Общий запрет/разрешение "Показать каталог деталей всем" устанавливается в Settings.

- Поиск осуществляется по модели телефона.

- Orders - каталог заказанных деталей данного сервиса.

- All details - свои детали и детали участников сайта, разрешённые ими к просмотру.

Handbook

- Соответствие модели телефона надписи на нём.

- Поиск - по каждому параметру.

Messages

- Содержит таблицу шаблонов сообщений.

- При прохождении TELEPHONES --> edit --> Send message из таблицы MESSAGES

  используеся наиболее подходящий шаблон, который затем редактируется и отсылается.

- Sent messages all - переход к таблице высланных сообщений.

- В данной версии программы используются только e-mail сообщения.

- Поиск - по совпадению слов с текстом сообщений.

Settings
